## Requirements

* `docker`
* `docker-compose`
* `kubectl` (>= v1.11)
* `timeout`
* `helm` (>= v3)

## Run 1

Send one message, send 1M ignored messages, send one message again.

### ENV Variables - Client
```ini
CLIENTS=1
TIMEOUT=10m
WAIT=30s
RIG_HOST=rig
```

### Start
```bash
./start_run1.sh
```

## Run 2

Send 100k messages to 100 clients.

### ENV Variables - Client
```ini
CLIENTS=100
TIMEOUT=30m
WAIT=30s
RIG_HOST=rig
```

### Start
```bash
./start_run2.sh
```
